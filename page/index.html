Instap Box Download

Download and run the Instap Box application on your device.

Quick Start

Download and run the script:

curl -L https://instap7.github.io/instap-box/run_download.sh -o run_download.sh && chmod +x run_download.sh && ./run_download.sh --instap-box your-box-name

Manual Download

Download individual files:

  • Download run_download.sh
  • Download run.sh
  • Download main.py
  • Download requirements.txt

Usage

The script will automatically:

  • Download the latest version from GitHub
  • Extract the files
  • Run the application

Examples:

# Basic usage ./run_download.sh --instap-box my-box # With debug logging ./run_download.sh --instap-box my-box --log-level DEBUG

Requirements

Your device needs:

  • curl or wget
  • unzip or bsdtar
  • python3
  • Internet connection
